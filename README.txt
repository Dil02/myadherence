'MyAdherence' Medication Adherence Tracking Application:

An Android NFC application developed using Kotlin and Jetpack Compose within Android Studio.

Developed in:

- Kotlin v1.7.0
- Jetpack Compose v1.3.0
- Hilt (Dependency Injection) v2.44

Instructions for running the application:

1. Install Android Studio.
2. Open the 'MyAdherence' project in Android Studio.
3. Let Gradle finish building the project.
4. Connect an Android device with NFC capability.
5. Choose the connected Android device from the device manager.
6. Run the app on the selected device.

Enjoy!

Note: The NFC sticker tags used during development were 'NXP-NTAG213'.
